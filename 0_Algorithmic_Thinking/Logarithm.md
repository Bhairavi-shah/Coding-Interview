Logarithms
==========

It says _Power to raise the base to, in order to get the answer_

log<sub>10</sub>(100) = 2
This is because 10<sup>2</sup> = 100


Rules
-----

#### Simplification
log<sub>b</sub>(b<sup>x</sup>) = x

#### Multiplication
log<sub>b</sub>(x*y) = log<sub>b</sub>(x) + 
log<sub>b</sub>(y)

#### Division
log<sub>b</sub>(x/y) = log<sub>b</sub>(x) - 
log<sub>b</sub>(y)

#### Powers
log<sub>b</sub>(x<sup>y</sup>) = y * log<sub>b</sub>(x)

#### Change of base
log<sub>b</sub>(x) = log<sub>c</sub>(x) / 
log<sub>c</sub>(b)
